## Garbage Classification Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/14f9f2e0ea4f41adafd7329e524943e8/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Garbage Classification Administrator, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Garbage Classification
5. Garbage Map
6. Personal Center
7. Change Password

**Garbage Classification Administrator Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. View Users
6. Garbage Classification Administrator
7. Garbage Classification
8. Garbage Types
9. Garbage Map

**Administrator Functions:**
1. Login
2. Change Password
3. Personal Information
4. User Management
5. Garbage Classification Administrator
6. Garbage Classification
7. Garbage Types
8. Garbage Map
